# Sarvadisha-News-Portal
This is the project which we have done at the END of the SEMESTER integrating all the topics which we have been taught. This is a news webpage showing realtime data for students and also has a map explorer.

Sarvadisa is a student‑focused news and dashboard platform designed to deliver personalized experience.  
It features dual login dashboards (student,general), a map explorer for  news and places.

Features
 Dual dashboards: Student and General
 Map explorer for local discovery
 Personalized news feeds
